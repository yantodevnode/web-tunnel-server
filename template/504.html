What happened?

The HyperText Transfer Protocol (HTTP) 504 Gateway Timeout server error response code indicates that the "Proxy Server", while acting as a gateway or proxy, did not get a response in time from the upstream Database Local Server that it needed in order to complete the request.

What can I do?

Make sure the Database Local Server connection is connected to the Proxy Server. Check internet connection speed in Database Local Server. Check if data fetched from Database Local Server is available. If the retrieved data is not available, the "Gateway Timeout" message will still appear.
